# CodeAlpha Basic Chatbot

## Objective
Develop a simple rule-based chatbot using Python that can interact with users by responding to common questions and greetings.

## Features
- Greets the user.
- Responds to common inputs like "hello", "hi", and "how are you".
- Tells its name when asked.
- Ends the conversation when the user types "bye".
- Handles unknown inputs with a default response.

## Technologies Used
- Python
- Google Colab

## Files
- Basic_Chatbot.ipynb

## Sample Conversation

You: Hello

Bot: Hello! How can I help you?

You: How are you?

Bot: I am fine. Thank you!

You: Your name

Bot: I am a Basic Python Chatbot.

You: Bye

Bot: Goodbye! Have a nice day.
